# Execution Contractors Directory Wireframe

## Layout: Offline-First Contact List
*Crucial: Must read entirely from SQLite `installers` table.*
**Components:**
- **Search/Filter Bar:** "Search plumbers, electricians..."
- **List Items:**
  - Avatar (Initials).
  - Name.
  - Expertise Area (Badge).
  - Standard Rate (e.g., "$150/day").
- **Trailing Action:** Native phone dialer icon to call the contractor immediately from the field.
- **FAB:** "Add Contractor" (Syncs to API, updates SQLite).
